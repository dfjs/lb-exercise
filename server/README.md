# Server

A basic (express) node server that consumes Bitstamp and CoinMarketCap HTTP APIs.

## Setup/environment requirements

- node 8

## Quick (dev) start:

Install and start server:

```
$ cd server/
$ npm i
$ npm start

```

See http://localhost:3000/ for API endpoints list.
